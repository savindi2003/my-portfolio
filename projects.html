M-Commerce Platform

My Bookings

MyBookings is an m-commerce hotel booking Android application that enables users to search and book hotels with ease. Built with Firebase Firestore as the backend, the app includes hotel listings, booking management, and secure payments through PayHere. It integrates Google Maps SDK and Google Places API to display and search hotels based on user location. Upon booking, the system generates a QR code (using ZXing) that can be scanned via Google ML Kit for quick check-in. The app also supports real-time updates and push notifications using Firebase Cloud Messaging, ensuring a smooth and efficient booking experience.

Java Firebase Firestore Firebase Cloud Messaging Google Maps SDK Google Places API ZXing Google ML Kit

Auction Web Application

Ezy Bid

Ezy Bid is a real-time online auction web application developed using Java EE (EJB, JMS, WebSocket, and JSP). The system allows users to register, log in, and participate in live auctions where bids are updated instantly across all clients. It is designed to handle high concurrency through Stateless Session Beans, while JMS Topics and Message Driven Beans ensure asynchronous bid broadcasting. Real-time updates are delivered to the user interface via WebSocket integration, providing a seamless bidding experience. The application follows a modular EAR architecture with separated Web, EJB, and Core modules, ensuring scalability, maintainability, and high performance under heavy load.

Java EE EJB JMS WebSocket JSP HTML CSS JavaScript

Core Banking System

National Bank

The National Bank Core Banking System is developed using Enterprise JavaBeans (EJB) with a strong focus on EJB Timer Services to automate time-sensitive banking operations. The system supports multiple user roles including Bank Teller, Customer Service Officer, Admin, and Super Admin, each with secure role-based access.

Core features include automated scheduled fund transfers, monthly interest calculation, daily balance updates, and automatic transaction report generation at scheduled intervals. Transaction consistency and reliability are maintained using Container-Managed Transactions, while Java Transaction API (JTA) ensures atomic operations for global transactions like fund transfers.

A robust security architecture is implemented using JAAS authentication and role-based authorization with Java EE security annotations (@RolesAllowed, @PermitAll, @DenyAll). To ensure reliability, the system uses persistent timers, connection pooling, and audit logging, making it highly scalable, secure, and resilient for 24×7 banking operations.

Java EE EJB EJB Timer Service Java Transaction API JAAS Security MySQL
Java E-Commerce Platform

ShoeSpere

ShoeSpere is a full-featured e-commerce web application developed with Java EE 7. The system provides core shopping functionalities including product listing, shopping cart, session-based cart (available even for non-logged-in users), checkout process, and PayHere payment gateway integration for secure online transactions.

The backend is built with Java EE Servlets and Hibernate ORM to manage database operations efficiently, ensuring data persistence and scalability. On the frontend, HTML, CSS (a template) and JavaScript are used to deliver a responsive and user-friendly shopping experience. Session management is implemented to allow users to add items to the cart before authentication, and seamlessly continue after logging in.

Java EE Servlets Hibernate ORM MySQL PayHere API JavaScript
IoT-based plant water monitoring system

HydroGrow

HydroGrow is an IoT-based plant water monitoring system that integrates hardware sensors, a backend server, and a mobile application to provide real-time plant health insights. The system consists of three main components:

🌱 Arduino + ESP32 – Collects soil moisture levels through sensors and transmits the data to the backend.
☕ Java Server (REST APIs) – Handles sensor data processing, stores data in the database, and provides APIs for mobile access.
📱 React Native App – User-friendly mobile interface to monitor soil moisture status, view plant health in real time, and track historical data stored in the backend.

This system allows users to continuously monitor their plants’ water requirements, ensuring efficient plant care through real-time IoT data collection and visualization.

React Native Java Arduino ESP 32
